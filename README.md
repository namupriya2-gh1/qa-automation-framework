# QA Automation Framework (Python + Playwright)
QA Automation Framework using Python, Playwright, Pytest, and API Testing (Work in Progress)


This project demonstrates a test automation framework built using Python and Playwright, along with API testing.

## Features
- UI Automation using Playwright
- API Testing using Python requests
- Test execution using pytest
- Page Object Model (POM) design pattern

## Tech Stack
- Python
- Playwright
- Pytest
- Requests

## Work in Progress
This project is an active QA automation framework for UI and API testing, built with Python and Playwright.
